## How to start

Prerequest:
- Ubuntu 16.04
- GCC installed

```
$ sudo chmod u+x test.sh
$ ./test.sh
```

## Detail Explaination
### CPU Info
`user`: normal processes executing in user mode

`nice`: niced processes executing in user mode

`system`: processes executing in kernel mode

`idle`: twiddling thumbs
